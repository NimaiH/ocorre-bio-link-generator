# 🚀 O Corre - Bio Link Generator

Ferramenta de automação para microempreendedores, focada em gerar páginas de links profissionais em segundos.

## 🛠️ Tecnologias
- Python & Streamlit (Dashboard)
- Scikit-Learn (Inteligência de cores via K-Means)
- HTML5/CSS3 (Templates dinâmicos)

## 🏃 Como rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o arquivo `run_dashboard.bat` ou use `python -m streamlit run app.py`

## 📈 Objetivo
Digitalizar negócios locais com baixo custo e alta eficiência estética.
